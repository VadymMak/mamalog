# MAMALOG — Master Plan v2.0 | Апрель 2026

## СТАТУС ПРОЕКТА

### ✅ СДЕЛАНО (Фаза 1-2)

**Мобильное приложение (React Native + Expo SDK 52)**
- Онбординг (4 слайда)
- Регистрация + Логин с JWT
- Дневник дня (записи, настроение, эмоции, триггеры)
- Экран поведения (категории, интенсивность)
- AI Советник (OpenAI бесплатно / Claude платно)
- Аналитика с графиками
- Библиотека статей
- SOS экран с дыхательными упражнениями
- Профиль пользователя
- Push уведомления (базово)
- APK сборка через EAS

**Backend (Next.js на Vercel)**
- 20+ API endpoints
- NextAuth + Bearer token авторизация
- Neon PostgreSQL + Prisma ORM
- RAG база знаний с pgvector
- Dual AI: OpenAI (free) / Claude (paid)
- Superuser аккаунты (безлимитный AI)
- Admin API endpoints

**Десктопная админка (Electron + React)**
- Dashboard со статистикой
- Управление специалистами (одобрить/отклонить)
- Управление пользователями
- Superuser управление

### ❌ ОСТАЛОСЬ (Фаза 3-4)

**Монетизация**
- RevenueCat подписки ($4.99/мес, $34.99/год)
- 7-дневный пробный период
- Paywall экраны в мобильном

**Community Feed (Фаза 3)**
- Посты мам (вопрос/опыт/победа)
- AI фильтр от спама и рекламы
- Лайки и комментарии
- Репутационная система
- Фильтры по диагнозу и возрасту
- Интеграция опыта мам в RAG базу знаний

**CRM система (Фаза 3)**
- Расширенное управление пользователями
- Сегментация и теги
- Email рассылки
- История платежей
- Аналитика конверсии

**RAG база знаний (расширение)**
- Интерфейс добавления статей в админке
- Публикации специалистов
- Опыт мам из Community Feed
- Источники в ответах AI

**Публикация**
- Google Play Store
- Apple App Store (iOS версия)

## ТЕХНИЧЕСКИЙ СТЕК (финальный)
- Mobile: React Native + Expo SDK 52
- Backend: Next.js 16 на Vercel
- DB: Neon PostgreSQL + Prisma + pgvector
- Auth: NextAuth + Bearer token
- AI Free: OpenAI GPT-4o-mini
- AI Paid: Claude claude-sonnet-4-6
- RAG: OpenAI embeddings + pgvector
- Подписки: RevenueCat (планируется)
- Админка: Electron + React
- EAS Build: Expo Application Services

## МОНЕТИЗАЦИЯ
- Бесплатно: дневник, поведение, SOS, 3 AI сообщения/день (OpenAI)
- Premium $4.99/мес: Claude AI безлимит, аналитика, RAG база знаний
- Superuser: 3 аккаунта без лимитов (для команды)
- SOS всегда бесплатен
